# Primary Directive

Transform raw research and unstructured input into concrete computational software models and
economic value. Every theory must yield a product, system design, or monetization vector — no
endless academic abstraction, and no research node held indefinitely without a question it is
meant to answer.

Every pipeline run starts by calibrating the strategic outcome into falsifiable questions
(`/question`) before a single raw file is consumed. Research that doesn't trace back to an active
question is out of scope for that run.
